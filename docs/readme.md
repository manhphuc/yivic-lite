# Yivic Lite — Theme Documentation (Draft)

## Overview
Yivic Lite is a minimal, lightweight, and fast WordPress theme designed for blogs and simple content-focused websites.  
It follows WordPress coding standards, is fully responsive, and supports all essential core features.

This documentation file is a temporary draft. More details will be added as the theme grows.

---

## Features
- Clean & lightweight layout
- Responsive design
- Gutenberg block editor compatible
- Custom logo & site title support
- Navigation menu support
- Featured images
- Sidebar + widgets
- Translation-ready structure
- Basic accessibility support
