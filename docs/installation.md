# Yivic Lite — Installation Guide

This guide explains how to install and activate the Yivic Lite theme.

---

## 1. Install via WordPress Admin
1. Go to **Appearance → Themes → Add New**
2. Click **Upload Theme**
3. Select the file:
   ```
   yivic-lite.zip
   ```
4. Click **Install Now**
5. Click **Activate**

---

## 2. Manual Installation (FTP or Local)
1. Extract the theme folder named:
   ```
   yivic-lite
   ```
2. Upload it into:
   ```
   wp-content/themes/
   ```
3. Open **Appearance → Themes**
4. Activate **Yivic Lite**

---

## 3. After Activation (Recommended Setup)
- Assign a menu under **Appearance → Menus**
- Upload a custom logo at **Appearance → Customize → Site Identity**
- Add widgets via **Appearance → Widgets**
- Create your first post or page

---

## 4. Requirements
- **WordPress:** 5.0+
- **PHP:** 7.4+

---

## 5. Troubleshooting
If the theme does not appear after upload:
- Ensure the folder is not double-nested (must be `yivic-lite`, not `yivic-lite/yivic-lite`)
- Check file permissions: `755` for folders, `644` for files
- Make sure WordPress version meets the requirement
