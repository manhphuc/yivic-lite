# Yivic Lite — How to Use the Theme

This document explains the basic usage of Yivic Lite.  
More advanced customizations will be added later as the theme evolves.

---

## 1. Customizing the Header
Go to:
```
Appearance → Customize → Site Identity
```
You can change:
- Site title
- Tagline
- Logo
- Site icon (favicon)

---

## 2. Setting Up the Navigation Menu
1. Go to **Appearance → Menus**
2. Create a new menu
3. Add pages, categories, or custom links
4. Assign it to the **Primary Menu** location

---

## 3. Using Widgets
Navigate to:
```
Appearance → Widgets
```

You can add:
- Search widget
- Recent posts
- Categories
- Custom HTML
- And more

Widgets will appear in your sidebar or footer depending on the layout.

---

## 4. Creating Posts & Pages
Yivic Lite supports:
- Gutenberg blocks
- Featured images
- Categories & tags

Simply create posts under:
```
Posts → Add New
```

---

## 5. Recommended Plugins (Optional)
Yivic Lite works without plugins, but you may consider:
- **Classic Widgets** (if you like old UI)
- **Regenerate Thumbnails**
- **SEO plugins (Yoast/RankMath)**

These are optional.
