== Description ==

Yivic Lite is a clean, minimalist, and lightweight WordPress theme crafted for bloggers, writers, and content-focused websites.

It follows WordPress core guidelines strictly, without including any unnecessary frameworks or custom builders. The design emphasizes usability, readability, and compatibility with both classic and block editors — making it ideal for simple blogs, personal branding, or small business sites that need a fast and reliable layout.

### Key Features:
* **Lightweight & Fast:** No frameworks, no heavy libraries — only core WordPress features.
* **Clean Typography:** Focus on readability and content clarity.
* **Full Gutenberg Support:** Works seamlessly with block-based content.
* **Custom Logo & Menus:** Standard customizer settings supported out-of-the-box.
* **Featured Images:** Perfect for blog posts and article thumbnails.
* **Responsive Layout:** Works smoothly on desktops, tablets, and mobile devices.
* **Translation-Ready:** Includes support for localization and `yivic-lite` text domain.
* **Accessible by Design:** Includes skip-link and semantic markup for better accessibility.

Yivic Lite aims to provide a simple, classic blogging experience with modern coding standards and long-term compatibility.

== Installation ==

### From the WordPress Dashboard:
1. Navigate to **Appearance → Themes → Add New**.
2. Search for **Yivic Lite**.
3. Click **Install**, then **Activate**.

### Manual Installation:
1. Upload the `yivic-lite` folder to `/wp-content/themes/`.
2. Go to **Appearance → Themes** and activate *Yivic Lite*.
3. (Optional) Assign your Primary Menu under **Appearance → Menus**.
4. (Optional) Customize your site title, logo, header, and layout under **Appearance → Customize**.

== Frequently Asked Questions ==

= Does Yivic Lite use any custom frameworks or page builders? =
No. The theme uses only native WordPress features to ensure maximum speed and compatibility.

= Is the theme compatible with the block editor (Gutenberg)? =
Yes. Yivic Lite works with Gutenberg and does not add custom blocks to avoid unnecessary bloat.

= Can I use the theme for commercial projects? =
Yes. The theme is licensed under GPLv3, so you can use it for personal or commercial websites.

= Where can I report issues or request enhancements? =
You can report issues via the WordPress.org support forum.
