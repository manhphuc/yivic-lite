== Description ==

Yivic Lite is a clean, minimalist, and lightweight WordPress theme crafted for bloggers, writers, and content-focused websites.

It follows WordPress core guidelines strictly, without including unnecessary frameworks or builders. The design emphasizes usability, readability, and long-term compatibility with both the classic editor and Gutenberg.

### Key Features
* **Lightweight & Fast:** No external frameworks or heavy assets.
* **Modern Typography:** Optimized for clarity and reading comfort.
* **Full Gutenberg Support:** Works seamlessly with block-based content.
* **Custom Logo & Menus:** Makes use of native Customizer features.
* **Responsive Layout:** Looks great on desktops, tablets, and smartphones.
* **Featured Images:** Ideal for blogs and news-style layouts.
* **Translation-Ready:** Includes full localization support.
* **Accessibility-Focused:** Includes skip-link and semantic HTML structure.

Yivic Lite aims to provide a straightforward, timeless blogging experience built on modern coding standards and high performance.

== Installation ==

### From the WordPress Dashboard
1. Navigate to **Appearance → Themes → Add New**.
2. Search for **Yivic Lite**.
3. Click **Install**, then **Activate**.

### Manual Installation
1. Upload the `yivic-lite` folder to `/wp-content/themes/`.
2. Go to **Appearance → Themes** and activate *Yivic Lite*.
3. (Optional) Assign menus under **Appearance → Menus**.
4. (Optional) Customize the site logo, title, and layout under **Appearance → Customize**.

== Frequently Asked Questions ==

= Does Yivic Lite use any custom frameworks or page builders? =
No. The theme uses only native WordPress features to ensure speed and long-term compatibility.

= Is the theme compatible with the block editor (Gutenberg)? =
Yes. The theme supports Gutenberg out of the box and does not add custom blocks to avoid unnecessary bloat.

= Can I use the theme for commercial projects? =
Yes. Yivic Lite is licensed under GPLv3, allowing use in personal and commercial projects.

= Where can I report issues or request new features? =
You can report issues through the WordPress.org support forum.
